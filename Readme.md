# Полезные материалы для руководителей

Здесь сообщество IT-руководителей собирает книги, фильмы, сериалы и подкасты,  
которые помогают расти в управлении — от тимлида до топ-менеджера.

Каждый материал сопровождается коротким объяснением, чем он полезен,  
и честной критикой — чтобы вы понимали контекст, а не просто получали список.

## Критерии отбора

Мы добавляем материал, если он:
- напрямую связан с управлением людьми, командами или организациями;
- содержит практическую пользу или нетривиальный взгляд на управление;
- подходит хотя бы одному уровню: тимлид, менеджер среднего звена, топ-менеджер.

Мотивационный контент без конкретики и «вода» — не берём.

## Как добавить свой материал

Вы можете предложить материал двумя способами — выбирайте удобный.

### Способ 1 — через Issue

1. Нажмите вкладку **Issues** вверху страницы.
2. Нажмите **New issue**.
3. Напишите в свободной форме:
    - название материала и автор/источник;
    - тип: книга / фильм / сериал / подкаст / статья;
    - кому подойдёт и чем полезно (2–3 предложения);
    - ваша критика или оговорки (честно приветствуется).
4. Нажмите **Submit new issue** — готово, мы разберём и добавим.

### Способ 2 — через Pull Request

1. Сделайте форк этого репозитория.
2. Отредактируйте `Readme.md` — добавьте строку в нужную таблицу.
3. Отправьте Pull Request с коротким описанием, что и зачем добавили.
