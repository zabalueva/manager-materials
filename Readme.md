# Полезные материалы для руководителей
Здесь мы собираем различные материалы, которые могут быть полезны руководителям разных уровней. Материалы разбиты по разным типам: книги, доклады, статьи и т.д.
Рекомендуем, объясняем почему это может быть полезно и собираем критику материала для более полной картины.

| Номер | Материал | Чем полезно | Критика |
| :--- | :---: | :---: | :---: |
|  | **Книги** |
| 1 | М. Уоткинс "Первые 90 дней. Стратегии успеха для руководителей всех уровней"  | Рассказывает как руководителю позаботиться о своей адаптации в новой команде / компании и объясняет огромную важность организации адаптации для руководителей. |
| 2 | М. Перескокова "Мама, я тимлид" | Хороший погружающий в профессию материал с примерами. Подойдет для начинающих тимлидов. |
| 3 | П. Друкер "Эффективный руководитель" | Классика менеджмента, которая показывает, что эффективность — навык, а не талант, и даёт практичные управленческие принципы. |
